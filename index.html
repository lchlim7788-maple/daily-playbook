∑ 这些价位是怎么算出来的

剧本不是预测涨跌,而是把"市场结构"摆出来:在哪些价位上方/下方,大概率会发生什么。所有数字由公式从前一日的高/低/收盘和波动率推导,逻辑透明、可复现。

轴心点(Pivot Points)— R / S 价位

用前一交易日的高(H)、低(L)、收盘(C)算出当日参考价位:

  • PP = (H + L + C) / 3 — 当日轴心,多空分界
  • R1 = 2·PP − L ，S1 = 2·PP − H — 第一道阻力/支撑
  • R2 = PP + (H−L) ，S2 = PP − (H−L) — 第二道,常作"伸展目标"
预期波动区间(Expected Move)— 金色虚线带

从 VIX 反推当日 ±1σ:当日σ ≈ VIX/100 ÷ √252，预期波动 = 参考价 × 当日σ。金色带约覆盖 68% 概率,带外即"超预期"。

Gamma 状态 — 为什么对 0DTE 最重要(本版尚未接入)

正 Gamma:做市商"涨了卖、跌了买"→ 抑制波动 → 倾向区间震荡、向关键行权价吸附。

负 Gamma:对冲方向反转 → 放大波动 → 倾向单边趋势、急涨急跌。

精确判断需要期权全链持仓量(OI)。本版(v1)未接入,状态显示"需 GEX 数据"。SPY/QQQ 链相对好拿,SPX 较难/需付费。

风险提示 · 本工具只展示基于技术分析与波动率的市场结构,不构成投资建议,也不是买卖指令。末日期权(0DTE)当日到期、时间价值衰减极快、Gamma 风险极高,可在数分钟内归零。技术价位会被跌穿/突破,情景推演经常落空。是否交易、如何控制仓位与止损,由你自行判断并承担后果。
